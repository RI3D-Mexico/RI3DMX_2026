# RI3DMX_2026

RI3D México busca apoyar a la comunidad de FRC en México y Latinoamérica mediante la creación, documentación y difusión de un robot funcional en los primeros tres días de la temporada posterior al kickoff.

Nuestro objetivo principal es demostrar nuestros procesos de ideación, diseño, manufactura, programación, estrategia, siendo así que podemos compartir:

- Código
- CAD
- Estrategia de juego
- Decisiones de ingeniería
- Lecciones aprendidas

Todo el contenido publicado y discutido es open-source y está pensado para servir como una referencia educativa para equipos rookies y veteranos. Asimismo, todo está enfocado desde la visión y cultura de Gracious Professionalism y Coopertition.
